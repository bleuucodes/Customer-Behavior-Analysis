**Customer Behavior Analysis**

**Overview**
Analyzed customer shopping behavior using transactional data to identify trends, customer segments, and key factors influencing purchasing decisions. The project focuses on improving business strategies through data-driven insights.

**Dataset**
- Records: 3,900
- Includes demographics, purchase details, discounts, reviews, and subscription data
- Missing values handled during preprocessing.

**Tools Used**
-SQL (PostgreSQL)
-Python (Pandas)
-Power BI
-CSV(Data)

**Work Done**
- Cleaned and transformed data using Python
- Performed SQL analysis to:
    -Segment customers (New, Returning, Loyal)
    -Analyze revenue, discounts, and product performance
    -Identify top products and customer trends
-Built an interactive Power BI dashboard

**Key Insights**
- Non-subscribers generate higher total revenue
- Certain products rely heavily on discounts
- Loyal customers form the largest segment
- Top products contribute significantly to sales

**Business Recommendations**
- Improve subscription offerings
- Focus on customer retention strategies
- Optimize discount usage
- Promote high-performing products

**Conclusion**
This project demonstrates end-to-end data analysis, from data cleaning to visualization, providing actionable insights for business decision-making.
